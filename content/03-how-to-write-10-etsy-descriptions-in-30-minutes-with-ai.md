You need three things:

1. **An AI tool** — ChatGPT, Claude, or any modern LLM (free tiers work fine)

## The Framework: One Prompt Structure, Infinite Variations

After testing hundreds of approaches, I landed on a single prompt structure that works for any product type. Here it is:

```
You are an expert Etsy copywriter who specialises in [CATEGORY] products.

Write a product description for this item:

PRODUCT: [Product Name]
FEATURES: [Material, dimensions, colour, customisation options]
PRICE: [$ amount]
TARGET CUSTOMER: [Who buys this?]

The description should:
- Be 3-4 short paragraphs (mobile-friendly)
- Start with a hook that paints a picture
- Include dimensions and materials naturally
- Address potential objections (quality, delivery, sizing)
- End with a clear call-to-action
- Use an [EMOTIONAL/PRACTICAL/MINIMALIST] tone

Also generate:
- A listing title (8-12 words, SEO-optimised, pipe-separated)
- 13 search tags (2-4 words each, no repeated words across tags)
- 3 key selling points in bullet format
```

**Example input:**

```
You are an expert Etsy copywriter who specialises in handmade ceramics.

Write a product description for this item:

PRODUCT: Handthrown Stoneware Coffee Mug
FEATURES: Speckled grey stoneware, approx 350ml capacity, microwave & dishwasher safe, each piece is unique
PRICE: £28
TARGET CUSTOMER: Coffee lovers who appreciate handmade items, gift shoppers

The description should:
- Be 3-4 short paragraphs
- Start with a hook that paints a picture of morning coffee
- Include dimensions and materials naturally
- Address "will it break in shipping?"
- End with a clear CTA
```

## The Workflow: 10 Descriptions in 30 Minutes

Here's the exact step-by-step I use:

**Minutes 0-5: Prepare your product spreadsheet**
List each product with: Name, 3-5 features, target customer, price. One row per product.

**Minutes 5-10: Batch-generate descriptions**
Copy the prompt framework above, paste your first product details, run it. Do 2-3 at a time in separate conversations (AI tools handle parallel chats). Takes 30 seconds per product.

**Minutes 10-15: Generate titles and tags**
Run each product through a SEO-focused prompt to generate titles and 13 tags. Cross-check for repeated words.

**Minutes 15-25: Review and personalise**
Read each description. Two things to check:
- Does it sound like YOU? Add your personality if needed
- Is it accurate? Verify dimensions, colours, materials

**Minutes 25-30: Final pass**
Read everything out loud. Fix anything that sounds robotic. Paste into Etsy.

## The "Voice" Layer: Making AI Sound Like You

The biggest fear sellers have about using AI is losing their voice. It's a valid concern.

Here's the trick: after the AI generates a description, do **one pass** where you swap out words that don't sound like you.

- If you're playful: add an exclamation mark, a joke, a casual line
- If you're serious: tighten the language, remove fluff
- If you're warm: add a personal detail, a story, a "why I make this"

This 60-second pass transforms generic AI copy into something that feels like it came from you. I do it for every single listing I write.

## What About Images, Policies, and Everything Else?

The same prompt approach works for your entire shop.

**Product Photography Brief:**
> "Generate a 10-shot product photo list for [product]. Include flat lays, lifestyle shots, detail close-ups, scale shots, and styled room photos. Specify ideal lighting and composition for each."

**Shop About Section:**
> "Write an 'About' section for an Etsy shop that sells [product type]. The maker started this shop because [reason]. Include why materials are chosen carefully and what makes their process unique. Warm, personal tone."

**Customer Message Templates:**
> "Write a friendly order confirmation message for a customer who just bought [product]. Include expected shipping date and a gentle review request. Keep it warm and professional."
